# Django Projects Portfolio

This repository contains a collection of Django-based web applications, showcasing various aspects of full-stack development, UI/UX design, and clean architecture.

## Projects

### 1. EstateAgency
A premium real estate agency template integrated with Django.
- **Key Features**: Property listings, agent profiles, contact forms, and a modern responsive UI.
- **Tech Stack**: Django, Bootstrap 5, FontAwesome, AOS (Animate on Scroll).

### 2. Project Gaurav

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/gauravphutane11/Django-Project-.git
   cd Django-Project-
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install Dependencies**:
   *(Note: Ensure you have Django installed in your environment)*
   ```bash
   pip install django
   ```

4. **Run a project**:
   Navigate into a project directory (e.g., `estate_agency`) and run:
   ```bash
   python manage.py runserver
   ```
